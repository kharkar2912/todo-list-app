# 📝 To-Do List App

A simple and interactive To-Do List web application that helps users manage their daily tasks efficiently.

## 🌐 Live Demo

👉 https://kharkar2912.github.io/todo-list-app/

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 💾 Saves tasks (if using local storage)
* 📱 Responsive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 How to Use

1. Open the live demo link
2. Enter your task in the input field
3. Click "Add"
4. Manage tasks easily

## 📦 Installation (Run Locally)

```bash
git clone https://github.com/kharkar2912/todo-list-app.git
cd todo-list-app
```

Open `index.html` in your browser.
